# AI Resume Analyzer 🚀

An intelligent web application that analyzes resumes and compares them with job descriptions using NLP techniques.

---

## 🔥 Features
- Upload Resume (PDF)
- Extract Resume Content
- ATS Score Calculation
- Skill Matching
- Missing Skills Detection
- Suggestions to Improve Resume

---

## 🛠 Tech Stack
- Python
- Streamlit
- NLP (spaCy / Transformers)
- PDF Parsing (pdfplumber)

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
